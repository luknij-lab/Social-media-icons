# Social-media-icons

<p>A lightweight and easy-to-integrate top bar module containing contact information and social media icons. Ideal as a starting point for WordPress-based websites or front-end projects. The bar displays phone numbers, email addresses, and active links to social media profiles, and its responsive design automatically adjusts the layout to fit smaller screens.</p>
<p>It can be used as a permanent information bar or as a secondary header to enhance the visibility of contact information. The code allows for easy customization of colors, fonts, and icons to match the design's style.</p>
